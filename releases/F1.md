# F1 Release — Working Catalog

## Summary

F1 — первая фаза разработки кода LKFL платформы. Результат: рабочий каталог льгот с multi-tenancy, авторизацией через Keycloak и RBAC.

**Дата релиза:** 2026-05-26  
**Тег:** `f1-complete`  
**Статус:** Hardening завершён

---

## Features

### Multi-tenant Architecture
- Tenant CRUD с валидацией slug и статусов
- Brand config (primary/secondary color, logo, favicon, CSS variables, meta tags)
- Tenant resolver middleware (по subdomain / header)
- Tenant isolation на уровне repository (все запросы фильтруются по tenant_id)

### Authentication & Authorization
- Keycloak OIDC integration (authorization code flow + implicit)
- JWT verification middleware
- RBAC: admin, hr, catalog_manager, employee
- Session management через Redis (TTL 24h)
- CSRF protection (state parameter)

### User Management
- Профиль сотрудника (GET/PUT /api/v1/users/me)
- Admin список пользователей с пагинацией, фильтрацией и поиском
- Admin обновление и деактивация пользователей
- Tenant isolation в админ-операциях

### Engagement Catalog
- Категории (CRUD через admin API)
- Engagement types (benefit, activity) с офферами
- Public API: список, детали, фильтрация по типу/категории/статусу
- Redis caching для списка каталога
- Cache invalidation при admin-записи

### Frontend SPA
- Vite + React 18 + TypeScript
- April UI (`@ukituki-ps/april-ui`) + Mantine
- Zustand для state management
- Страницы: /catalog, /dashboard, /login, /admin/*

### Testing
- Unit tests: Go (backend) + Vitest (frontend)
- Integration tests: testcontainers (PG + Redis)
- E2E tests: Playwright (Chromium, Firefox, Webkit)
- Chaos tests: 100 тестов отказоустойчивости
- Load testing: k6

### Infrastructure
- Docker production images (multi-stage, distroless, cosign)
- Prometheus + Grafana monitoring
- Loki + Promtail logging
- CI/CD: GitHub Actions (9 jobs)
- Staging deployment: docker-compose.staging.yml

### Security
- OWASP Top 10 audit passed
- Rate limiting на auth/catalog/admin endpoints
- CORS policy configured
- Dependency audit: govulncheck, npm audit, trivy

- **No billing/flow/consent** — запланировано в F2 (M23–M29)
- **No activity/survey/gamification** — запланировано в F3 (M30–M38)
- **No LLM/payments/recommendations** — запланировано в F4 (M39–M44)
- **No mobile app** — запланировано в F4

## Architecture

### Backend
- **Language:** Go 1.23+
- **Architecture:** Modular monolith (17 internal packages)
- **Router:** chi/v5
- **Database:** PostgreSQL 17 (schema `lkfl_platform`)
- **Cache:** Redis 7
- **Auth:** Keycloak 25 (OIDC)
- **Background jobs:** Asynq

### Frontend
- **Framework:** React 18
- **Build:** Vite
- **UI:** April UI + Mantine
- **State:** Zustand
- **API:** fetch через Nginx `/api/v1/`
- **Types:** openapi-typescript codegen

### Infrastructure
- **Container:** Docker (multi-stage, distroless)
- **Orchestration:** docker-compose
- **Reverse proxy:** Nginx
- **Monitoring:** Prometheus + Grafana
- **Logging:** Loki + Promtail
- **CI/CD:** GitHub Actions

2. Обновите Docker images
