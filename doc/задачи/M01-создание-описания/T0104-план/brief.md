# T0104 — Формирование раздела «План»

## Веха

M01-создание-описания

## Раздел

`doc/план/`

## Контекст

Раздел «План» описывает milestones, задачи, зависимости и exit criteria. На его основе создаются рабочие задачи в `задачи/`. Сейчас в `doc/план/` только README.md. Нужно создать полноценные файлы раздела.

## Источник данных

- **Кубики** — таймлайн: запуск лето 2026, полная работа август 2026
- **Функциональные требования** — для определения объёма работ
- **Прототип** — для оценки frontend-работ
- **Результаты T0101-T0103** — контекст, архитектура, спецификация

## Что сделать

Создать 4 файла в `doc/план/`:

### 1. `milestones.md`

Перечень вех проекта:

| Веха | Описание | Дедлайн |
|------|----------|---------|
| M00 | Setup, документация | май 2026 |
| M01 | Создание описания | май 2026 |
| M02 | Foundation (инфраструктура) | июнь 2026 |
| M03 | Auth & Users | июнь 2026 |
| M04 | Benefits catalog | июль 2026 |
| M05 | ДМС wizard | июль 2026 |
| M06 | Админ-панель | август 2026 |
| M07 | Интеграции | август 2026 |
| M08 | Тестирование | сентябрь 2026 |

### 2. `tasks.md`

Реестр задач по вехам:

| Веха | Задача | Описание | Статус |
|------|--------|----------|--------|
| M01 | T0101 | Контекст | в работе |
| M01 | T0102 | Архитектура | в работе |
| M01 | T0103 | Vision | в работе |
| M01 | T0104 | План | в работе |

(После M01 — добавить задачи по реализации)

### 3. `dependencies.md`

Граф зависимостей:

```
M01 → M02 → M03 → M04 → M05 → M06 → M07 → M08
  │       │     │     │     │     │     │
  └─документация────→ инфраструктура → разработка → тестирование
```

Зависимости между задачами:
- T0102 зависит от T0101 (нужен контекст для архитектуры)
- T0103 зависит от T0101, T0102 (нужен контекст и архитектура для спецификации)
- T0104 зависит от T0101, T0102, T0103 (нужна вся документация для плана)

### 4. `критерии-вех.md`

Exit criteria для проекта:

| Критерий | Описание | Веха |
|----------|----------|------|
| Документация | Все разделы заполнены | M01 |
| Инфраструктура | Docker, миграции, конфиг | M02 |
| Auth | Регистрация, авторизация, JWT | M03 |
| Каталог | CRUD льгот, фильтры, поиск | M04 |
| ДМС | Wizard, полисы, клиники | M05 |
| Админ | Импорт, отчёты, управление | M06 |
| Интеграции | ready4.ru, PrimeZone, кадровая | M07 |
| Тесты | Unit + E2E, coverage >80% | M08 |
