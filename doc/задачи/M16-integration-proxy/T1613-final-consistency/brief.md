# T1613 — Final consistency check

## Веха

M16-integration-proxy

## Контекст

После всех изменений нужна финальная проверка консистентности.

## Что сделать

1. Проверить все ссылки на `internal/integrations/` — должны быть либо удалены, либо изменены на `integration-proxy/`
2. Проверить все ссылки на `ProviderGateway` — должны указывать на gRPC client
3. Проверить все упоминания "1 бинарник" — должны быть обновлены на "2 бинарника"
4. Проверить ADR-024 — должна быть секция об исключении для proxy
5. Проверить NAVIGATION.md — все ссылки актуальны
6. Проверить спецификация/api.md — backend mapping актуален

### Файлы-мишени

| Действие | Файл |
|----------|------|
| Проверить | Все файлы в `doc/` |

### Критерии приёмки

- [ ] 0 ссылок на `internal/integrations/` как пакет монолита
- [ ] ProviderGateway → gRPC client везде
- [ ] "1 бинарник" → "2 бинарника" (где применимо)
- [ ] ADR-024 обновлён
- [ ] NAVIGATION.md актуален
- [ ] api.md backend mapping актуален
- [ ] 0 рассинхронов
