## Что сделано

### `src/pages/Dashboard.tsx` — полная реализация

- **Приветствие**: `{greeting}, {first_name}!` — имя берётся из `GET /api/v1/users/me` через `getUserProfile()`.
- **Временное приветствие**: 4 варианта по часам суток (Доброй ночи / Доброе утро / Добрый день / Добрый вечер).
- **3 stat card placeholder**: «Баланс баллов», «Активные льготы», «Доступные льготы» — значения «—», подключатся в F2.
- **Event feed placeholder**: карточка «Последние события» с текстом-заглушкой.
- **Quick actions**: ссылки-карточки на `/catalog`, `/documents`, `/support`.
- **Loading state**: `<Loader />` по центру экрана при загрузке профиля.
- **Error state**: сообщение «Ошибка загрузки профиля» при ошибке запроса.

### Архитектура

- Данные профиля загружаются через `useQuery` (@tanstack/react-query) + `getUserProfile()` из `@/api/user`.
- Компоненты `StatCard`, `QuickActionLink`, `getGreeting` вынесены в отдельные функции внутри файла.
- JSDoc комментарии для публичных компонентов и утилит.
- Стилизация inline (согласно паттерну проекта — см. Catalog.tsx).
