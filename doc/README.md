# Документация — Платформа гибких льгот

Единая корпоративная платформа для управления льготами 100 000+ сотрудников.
White-label, multi-tenant. Go backend + React frontend.

## Архитектура в двух словах

**Modular monolith + Integration Proxy.** Два бинарника (`lkfl-server` + `lkfl-integration-proxy`), один `go.mod`, 16 internal-пакетов монолита (14 business + tenant + api + integrationclient).
Межмодульная коммуникация в монолите — Go interfaces (compile-time type safety). Монолит ↔ proxy — gRPC (localhost). NATS не используется.
[M12: ADR-024] [M16: ADR-035]

```
┌─────────────────────────────────────────────────────────────┐
│  lkfl-server (:8080)                                        │
│                                                             │
│  16 internal-пакетов:                                       │
│  tenant/  auth/  user/  consent/  cel/  llm/  eligibility/  │
│  compliance/  engagement/ {catalog/ flow/ collections/ survey/}  │
│  notification/  gamification/  billing/  integrationclient/ │
│  payments/  content/  api/  recommendations/ (stub, M15)   │
│                                                             │
│  + shared/pkg/auth/  + shared/pkg/celcontext/              │
└──┬───────────────┬───────────────┬─────────────────────────┘
   │               │               │
 PostgreSQL      Redis           Keycloak
(PG 17)        (key prefixes)    (OIDC IdP)

┌──────────────────────────────────────┐
│  lkfl-integration-proxy (:8090)     │
│  adapters/  circuitbreaker/          │
│  webhook/  grpc/  config/            │
└──┬───────────────────────────────────┘
   │
  Внешние провайдеры (REST/HTTP)
```

### Frontend

React SPA: Vite + React 18 + `@ukituki-ps/april-ui` + `@ukituki-ps/april-tokens` + Mantine.
State management: Zustand. API: `fetch` через Nginx `/api/v1/`.

Архитектура фронтенда задокументирована: [`архитектура/фронтенд.md`](./архитектура/фронтенд.md), [`архитектура/фронтенд-mobile-forms.md`](./архитектура/фронтенд-mobile-forms.md).
