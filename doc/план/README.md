# План

> **🗺️ Навигация:** [`doc/NAVIGATION.md`](../NAVIGATION.md) — карта «вопрос → файл:строка»

Этот раздел описывает **milestones**, **задачи**, **зависимости** и **exit criteria** проекта.

## Назначение

План переводит Спецификацию в конкретные вехи и задачи. Он отвечает на вопросы:
- **В каком порядке выполняем работу?** — milestones M00 → M16
- **Какие задачи в каждой вехе?** — T{MM}{NN} с описанием и критериями
- **Что зависит от чего?** — dependency graph между задачами
- **Когда веха выполнена?** — exit criteria для каждой вехи

## Поток информации

```
План
  │  Milestones → Задачи         (родительская веха)
  │  Tasks      → Задачи         (создание brief.md)
  │  Depends    → Задачи         (depends_on)
  │  Exit crit. → Задачи         (критерии приёмки)
```

## Содержимое раздела

| Файл | Описание | Статус |

| [`задачи.md`](./задачи.md) | Реестр всех задач T{MM}{NN} со статусами | ✅ |

## Структура вехи

Каждая веха M{MM} содержит:
- `M{MM}-{slug}/` — папка вехи в `задачи/`
  - `overview.md` — описание вехи, цели, критерии
  - `T{MM}{NN}-{name}/` — папки задач
    - `brief.md` — контекст и план
    - `plan.yaml` — checklist выполнения
    - `report.md` — отчёт о реализации
