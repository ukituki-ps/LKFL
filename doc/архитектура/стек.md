## TL;DR (для агентов)

> Таблица всех технологий с версиями. 150 строк.
> - **Backend (Go):** строка 10 | **Frontend (React):** строка 40

## Краткое описание

Таблица всех технологий с версиями и обоснованием выбора. Основана на существующем `go.mod`, артефактах April экосистемы и архитектурных решениях (ADR 001–035).

**Бинарники:** 2 бинарника (`lkfl-server` + `lkfl-integration-proxy`), 1 `go.mod`. Межмодульная коммуникация: Go interfaces (монолит) + gRPC (localhost, mono ↔ proxy). **ADR-035.**

## Observability — стек мониторинга

ФТ §9: «Logging всех событий и операций». ФТ §5: «Резервирование БД, балансировка нагрузки».

### Метрики (Prometheus)

| Метрика | Тип | Описание |
|---------|-----|----------|
| `http_requests_total` | Counter | RPS по endpoint |
| `http_request_duration_seconds` | Histogram | Latency по endpoint |
| `billing_transactions_total` | Counter | Транзакции: `{type="credit"}` / `{type="debit"}` |
| `billing_balance_query_total` | Counter | Запросы баланса (peak при начале периода) |
| `cel_generation_total{status="success"\|"error"\|"drift"}` | Counter | **ADR-021:** LLM CEL генерации |
| `cel_evaluation_total{domain="billing"\|"eligibility"\|"flow"\|"recommendations"}` | Counter | **ADR-021:** CEL evaluation вызовы по доменам |
| `cel_evaluation_duration_seconds{domain}` | Histogram | **ADR-021:** Latency CEL evaluation |
| `llm_requests_total{agent="cel-generator"\|"moderation"\|"analytics"}` | Counter | **ADR-022:** LLM запросы по агенту |
| `llm_tokens_total{type="input"\|"output", model}` | Counter | **ADR-022:** Token usage |
| `llm_cost_today_usd{tenant}` | Gauge | **ADR-022:** Cost per tenant |
| `integration_provider_latency_seconds` | Histogram | **ADR-035:** Latency вызовов по провайдеру (proxy) |
| `integration_provider_errors_total` | Counter | **ADR-035:** Ошибки по провайдеру (proxy) |
| `integration_circuit_breaker_state` | Gauge | **ADR-035:** Состояние circuit breaker: 0=closed, 1=open, 2=half-open |
| `integration_sync_duration_seconds` | Histogram | **ADR-035:** Время синхронизации каталога по провайдеру |
| `integration_webhook_total` | Counter | **ADR-035:** Входящие webhooks по провайдеру |
| `integration_dead_letters_total` | Counter | **ADR-035:** Dead letters по провайдеру |
| `asynq_jobs_completed_total` | Counter | Завершённые фоновые задачи |
| `internal_package_calls_total{from, to}` | Histogram | **M12:** latency между internal пакетами (billing, integrations, payments) |
